Инструкция по запуску

    Для командной строки:
    
        Для Windows:
            
            1. Перейдите в главную директорию проекта
            2. Введите в консоль следующую команду
                python src\main.py
    
    Для IDE:
        1. Откройте файл root/src/main.py
        2. Запустите его
